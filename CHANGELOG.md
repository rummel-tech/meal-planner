# Changelog

All notable changes to Meal Planner will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [1.0.0] - 2026-04-03

### Added
- Meal planning and scheduling
- Recipe management with nutritional information
- Grocery list generation
- Meal logging and tracking
- User authentication
- API integration with backend service

### Changed
- Environment is now injected at build time via `--dart-define=ENVIRONMENT=production` instead of being hardcoded
- Error messages shown to users are now friendly and do not expose internal exception details
- Dart SDK constraint updated to support Dart 3 (`<4.0.0`)
